# Rewind-233
将B站弹幕中大于等于四个连续的“哈”替换为相应数量的“2333”。

## 安装：https://greasyfork.org/zh-CN/scripts/526127

曾经弹幕池中常见的233，随着新生代的到来似乎逐渐销声匿迹了。对一部分人来说，如今满屏的“哈哈哈”既令人唏嘘，又影响视频有效面积，脚本因此诞生。

233来源于猫扑表情第233号，是一张捶地大笑的表情，常用来表示哈哈大笑的意思。
